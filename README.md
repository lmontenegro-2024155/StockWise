# StockWise

StockWise es una aplicación web para la gestión de inventarios desarrollada con una arquitectura de microservicios.

##  Funcionalidades

- Autenticación con JWT.
- Gestión de productos (CRUD).
- Control de entradas y salidas de inventario.
- Alertas de bajo inventario y productos agotados.
- Reportes del inventario.

## Tecnologías

- .NET 8
- Node.js
- Express.js
- MongoDB
- pnpm

## Instalación

```bash
git clone <URL_DEL_REPOSITORIO>
cd StockWise
pnpm install
```

## Ejecutar

```bash
pnpm dev
```

## Servicios

- **Auth Service:** autenticación de usuarios.
- **Servicio A:** gestión del inventario.
- **Servicio B:** alertas y reportes.
